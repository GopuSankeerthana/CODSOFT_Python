# Password Generator

## Overview
The Password Generator is a simple Python application that generates strong and random passwords based on the length specified by the user. It uses a combination of uppercase letters, lowercase letters, numbers, and special characters to create secure passwords.

## Features
* User-defined password length
* Generates random and secure passwords
* Includes:
  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Special characters (!, @, #, $, %, etc.)
* Simple command-line interface

## Technologies Used
* Python
* Random Module
* String Module

## How It Works
1. The user enters the desired password length.
2. The program combines letters, digits, and special characters into a character pool.
3. A random password is generated using the specified length.
4. The generated password is displayed on the screen.
