# 📝 To-Do List Application

##  Description

The To-Do List Application is a simple Python program that helps users manage their daily tasks efficiently. The application provides a menu-driven interface where users can add, view, update, delete, and mark tasks as completed.

This project demonstrates the use of Python fundamentals, including lists, dictionaries, loops, conditional statements, and user input handling.

##  Objective

To develop a task management system that allows users to organize and track their tasks effectively.

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Mark tasks as completed
* Display task status (Pending/Completed)
* User-friendly command-line interface

## Technologies Used
* Python
* Jupyter Notebook

## 🚀 How to Run
1. Open the notebook in Jupyter Notebook or Google Colab.
2. Run all cells.
3. Follow the menu options displayed on the screen.
4. Manage tasks by adding, updating, deleting, or marking them as completed.
