# CodSoft Python Programming Internship

## Overview
This repository contains the projects completed during the **CodSoft Python Programming Internship**. The tasks focus on fundamental Python programming concepts such as user input handling, conditional statements, loops, functions, and Python libraries.

## Internship Tasks

### Task 1: To-Do List Application
**Objective:** Create a simple To-Do List application that allows users to manage their daily tasks.

**Features:**
* Add tasks
* View tasks
* Remove completed tasks
* Simple and user-friendly interface

**Files:**
* To_Do_list.ipynb
* README.md
---

### Task 2: Simple Calculator
**Objective:** Design a calculator that performs basic arithmetic operations.
**Features:**
* Addition
* Subtraction
* Multiplication
* Division

**Files:**
* Task2_Calculator.ipynb
* README.md
---

### Task 3: Password Generator
**Objective:** Generate strong and random passwords based on the user-specified length.
**Features:**
* User-defined password length
* Random password generation
* Includes letters, numbers, and special characters
*
